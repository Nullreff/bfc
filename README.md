Brainfuck Compiler
==================

This is a [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter/compiler written in C.

Building
--------

You'll need [CMake](http://www.cmake.org/).

```bash
cmake src/
make
```

This will produce a `bfc` executable in the current directory.

Usage
-----

See `bfc help`
